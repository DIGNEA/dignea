Detailed Description

template<typename OptProblem, typename InstSolution>
class MEAProblem< OptProblem, InstSolution >

MEA Instance Generation Problem. Abstract Problem for the MEA. This problem will be used as a superclass for any new instance generation problem. This is the type of problem for the MEA, not for the evaluation phase. It acts as a wrapper for the MEA.
